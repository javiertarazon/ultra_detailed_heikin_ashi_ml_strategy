## 🔧 INSTALACIÓN RÁPIDA

```bash
git clone https://github.com/javiertarazon/ultra_detailed_heikin_ashi_ml_strategy.git
cd ultra_detailed_heikin_ashi_ml_strategy
git checkout v4.5
```

### 2. Crear Virtual Environment
```bash
python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux/Mac
```

### 3. Instalar Dependencias
```bash
pip install -r requirements.txt
```

### 4. Configurar (Opcional)
```bash
# Ver configuración
cat descarga_datos/config/config.yaml

# Editar si es necesario
# (Sandbox habilitado por defecto)
```

### 5. Ejecutar Live Trading
```bash
python descarga_datos/main.py --live
```
